## 📱 What is FakeCall?

FakeCall is an open source app that lets you simulate incoming calls on your Android phone. It shows a realistic call screen so you can create fake calls whenever you want. The app uses a modern design style called Material 3 UI and supports Monet, which changes colors based on your phone’s wallpaper.

You can use FakeCall to:

- Avoid awkward situations by pretending you have a call.
- Practice answering calls.
- Test other apps that deal with phone calls.
- Create fun pranks using fake calls.
